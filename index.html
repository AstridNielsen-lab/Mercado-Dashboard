📦 Como Instalar (Instalador Completo):

  1. Baixe o arquivo Mercado-Neural-Setup-v1.0.0.exe
  2. Execute o instalador como administrador
  3. Siga o assistente de instalação
  4. Escolha as opções desejadas (atalhos, etc.)
  5. Clique em "Instalar" e aguarde a conclusão
  6. Execute pelo Menu Iniciar ou atalho na área de trabalho

💻 Execução via Linha de Comando:

"C:\Program Files\Mercado Neural\Mercado-Neural.exe"

🗑️ Para Desinstalar:

Painel de Controle → Programas → Mercado Neural → Desinstalar

📁 Como Usar (Versão Portable):

  1. Baixe o arquivo Mercado-Neural-v1.0.0-Windows-Portable.zip
  2. Extraia para uma pasta de sua escolha
  3. Execute Executar-Mercado-Neural.bat ou Mercado-Neural.exe
  4. Aguarde o carregamento (3-5 segundos)

💻 Execução via Linha de Comando:

cd "pasta-extraida"

.\Mercado-Neural.exe .

📋 Requisitos:

• Windows 10/11 (64-bit)
• 4GB RAM (recomendado: 8GB)
• 500MB espaço livre
